About

DezKVM-Go Viewer is a web-based interface for the DezKVM-Go mini KVM over USB device, allowing portable KVM access to remote systems via a simple and intuitive web interface.

It is designed to work seamlessly with the DezKVM-Go hardware, providing features such as video streaming, audio output, and keyboard/mouse input forwarding.
